# 花都医院文书格式 Skill

This folder contains a Codex skill for applying `广州市花都区人民医院文书格式要求（2023年版）`.

## Files

- `SKILL.md`: the complete workflow, formatting rules, DOCX implementation guidance, and validation checklist.
- `README.md`: this short usage note.

## Usage

Use this skill when you need to standardize a supplied Word, DOCX, PDF, or text draft to the hospital's 2023 document format. The expected workflow is to read the source file, preserve substantive content, apply the format rules in `SKILL.md`, and verify the final rendering before delivery.

To install it for automatic discovery, copy the `huadu-hospital-document-format` folder into your Codex skills directory, such as `%USERPROFILE%\.codex\skills`.
